# 🎁 Gift Nest

> A website that makes it easier to select the perfect gifts for your loved ones 💕

---

## About

**Gift Nest** is a thoughtfully designed gift discovery website that helps you find the right gift for any occasion. Whether it's a birthday, anniversary, or just a way to say you care — Gift Nest takes the stress out of gift shopping by presenting curated options in a clean, easy-to-browse interface.

---

## Features

- 🛍️ **Browse Gift Products** — Explore a curated catalog of 14+ gift items with images
- 💰 **Price Guide** — Built-in pricing reference to help you stay within budget
- 📋 **Instructions & Tips** — Guidance to help you choose the most thoughtful gift
- 📱 **Responsive Design** — Clean UI that works across devices
- ⚡ **Lightweight & Fast** — Pure HTML, CSS, and JavaScript — no frameworks needed

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/abishaha/Gift-nest.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Gift-nest
   ```

3. **Open in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to launch it in your default browser.
